What This App Does

This project is a Laravel 10 examination management application. It lets administrators create institutions and assign users, lets institution users manage students, grades, events and exam registrations, lets CCD/content users build exam content, and exposes API endpoints for exam start, result upload, and institution/event synchronization.

The core flow is: create an institution, assign a user, create exam content and course sessions, add questions, create an event, attach course sessions to the event, register students, start the exam, persist attempts in a per-exam JSON tracking file, evaluate the exam, then view or download results.

Important Helpers and Conventions

  • currentUser() returns the authenticated user.
  • currentInstitution() returns the route-bound institution.
  • instRoute() builds institution-prefixed route names such as institutions.exams.index.
  • paginateFromRequest() applies request-driven pagination.
  • successRes() and failRes() wrap workflow results in App\Support\Res.
  • Institution::getRouteKeyName() uses code, so institution URLs use codes rather than numeric ids.
