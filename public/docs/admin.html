Access and Layout

Admin pages use prefix admin/, route name prefix admin., and middleware auth plus admin.user. The current VerifyAdminUser middleware checks that a user is authenticated; User::isAdmin() compares the user email to config('app.admin.email').

Admin views live under resources/views/admin and use admin/layout.blade.php, admin/header.blade.php, and admin/_sidebar.blade.php.

Related Models

  • User is a Laravel authenticatable model with Sanctum tokens available and an isAdmin() helper.
  • Institution stores institution metadata and uses code as the route key. Global admins can set license_cost; self-service institution creation uses the default 300.
  • InstitutionUser links users to institutions, stores an admin or staff role via InstitutionUserRole, and is checked by institution middleware.
